# Backend

Este diretório contém o código do backend do sistema financeiro.

## Estrutura de Pastas

- **controllers/**: Controladores que gerenciam as requisições HTTP
- **models/**: Modelos de dados e entidades
- **routes/**: Definição de rotas da API
- **services/**: Lógica de negócios e serviços
- **middleware/**: Middlewares para autenticação, validação, etc.
- **config/**: Configurações da aplicação
- **utils/**: Funções utilitárias e helpers
